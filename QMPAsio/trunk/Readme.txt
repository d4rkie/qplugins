Notes on building QMPASIO
=========================

Ths ASIO SDK must be obtained from Steinberg as it is copyrighted / licensed material.
It is free to all developers and is located on: http://www.steinberg.net/324_1.html

References to locations for the ASIO SDK and the QCD/QMP PDK are defined by local
environment variables as follows:


	ASIOSDK refers to the root of the ASIO SDK (Ex: D:\Steinberg\asiosdk2.1)

	QCDPDK refers to the root of the QCD/QMP PDK (Ex: D:\QCDPDK)
